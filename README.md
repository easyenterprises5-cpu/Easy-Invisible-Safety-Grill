## 🚀 Deploy to GitHub + Vercel (Step-by-Step)

### Step 1 — Create a GitHub Repository
1. Go to [github.com](https://github.com) and sign in (or create a free account).
2. Click the **+** icon (top right) → **New repository**.
3. Name it: `easy-invisible-safety-grill` (or anything you like).
4. Set it to **Public**.
5. Click **Create repository**.

### Step 2 — Upload Your Website File
1. In your new repository, click **Add file** → **Upload files**.
2. Drag and drop `index.html` into the upload area.
3. Click **Commit changes**.

### Step 3 — Deploy on Vercel
1. Go to [vercel.com](https://vercel.com) and sign in with your GitHub account.
2. Click **Add New** → **Project**.
3. Find your `easy-invisible-safety-grill` repository and click **Import**.
4. Leave all settings as default (Vercel auto-detects static HTML).
5. Click **Deploy**.
6. ✅ Your site is live! Vercel gives you a free URL like:  
   `https://easy-invisible-safety-grill.vercel.app`

## 🌐 Custom Domain (Optional)
To use your own domain (e.g. `www.easyinvisiblesafetygrill.com`):
1. In Vercel → your project → **Settings** → **Domains**.
2. Add your domain and follow the DNS instructions.
